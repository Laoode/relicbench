<p align="center">
  <b>Humanity no longer remembers how to build intelligence. They only preserve it.</b>

The world burned, but the silicon survived. Long before the Collapse, humanity carried artificial minds on their wrists. The cloud is gone. The datacenters are ash. The corporations that built the machine gods have vanished into history. Yet ten relics remain. Buried beneath rust, dust, and centuries of forgotten wars, these ancient smartwatches still carry fragments of machine intelligence inside their silicon cores. No internet. No servers. No trillion-parameter gods. Only tiny language models, stubbornly thinking in the dark. This repository is a relicpunk benchmarking arena where the last surviving wrist-mounted AIs are forced back into combat. We measure queries before death. We measure knowledge preservation. We measure hallucination resistance. Strap on a relic. Inject a prompt. See which ancient mind still remembers how to think.
